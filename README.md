TranslateApi
==========

Translate 1.0.0 for PHP

This package is a wrapper of [Translate Yandex](https://tech.yandex.com/translate/doc/dg/concepts/About-docpage/) adapted to work with PHP.


Installation
============

```php
    composer require "haotx/translate-api:1.*"
```
